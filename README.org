* Read First
This is the navigation hub. New to the repo? Read this, then ARCHITECTURE.org.
ScribeShift (repo codename =concre=) is an AI content-generation and
social-scheduling platform.

* Project Structure
#+BEGIN_EXAMPLE
concre/
├── src/            Vite + React frontend (components, hooks, lib, styles)
├── server/         Express backend (routes, services, config, middleware, templates)
├── api/            Vercel serverless entry + cron jobs (check-posts, daily-reminder, refresh-metrics)
├── supabase/       Postgres migrations
├── public/         Static assets
└── dist/           Vite build output (generated)
#+END_EXAMPLE

* Quick Start
1. =npm install=
2. Copy secrets into =.env= at the repo root (never commit it — see CLAUDE.md).
3. Frontend dev server: =npm run dev= (Vite).
4. Backend: =node server/index.js= (Express API).
5. Background worker: =node server/worker.js= (scheduled posts / metrics).
